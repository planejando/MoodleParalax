Moodleparallax

Parallax Course Format - Formato de Curso Parallax

View project on GitHub

Project objective:

The Moodle Parallax Project is a course format who show the sessions in parallax format, allowing the creation of courses with a more nicer look.

O Projeto Moodle Parallax, é um formato de curso que exibe as sessões no formato Parallax, permitindo a criação de cursos com um visual mais agradável.

Compatibility:

The course format Moodle Parallax is compatible with versions 2.5, 2.6 , 2.7, 2.8 , 2.9 and 3.0

O formato de curso Moodle Parallax é compatível com as versões 2.5, 2.6, 2.7, 2.8, 2.9 e 3.0.

Author and Maintainer:

Este projeto é mantido por Fernando Bosco Santana (@planejando)Brazilian, Deesenvolvedor WEB since 2009 and working with Moodle since 2014.
